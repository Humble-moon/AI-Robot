# 智扫通机器人智能客服系统 - 简历亮点

## 项目概述

智扫通机器人智能客服系统是一个基于大语言模型（LLM）的智能问答系统，结合RAG（检索增强生成）和Agent（智能体）技术，为用户提供扫地机器人相关的实时问答、故障排查、使用建议和个性化报告生成服务。

## 技术栈

- **核心语言**：Python 3.10+
- **AI框架**：LangChain（智能体构建、工具集成）
- **模型服务**：通义千问（ChatTongyi、DashScopeEmbeddings）
- **向量存储**：Chroma DB（文档向量化、相似性检索）
- **前端交互**：Streamlit（实时聊天界面、流式输出）
- **工具库**：Pandas（数据处理）、PDF/TXT解析器（文档处理）
- **配置管理**：YAML、环境变量
- **部署方式**：本地部署、容器化部署

## 技术亮点

### 1. 模块化架构设计
- 清晰分离智能体、RAG服务、向量存储、模型工厂等核心组件
- 采用工厂模式管理模型实例，提高代码可维护性
- 模块化设计便于功能扩展和代码维护

### 2. 高级AI技术应用
- **RAG技术**：结合文档知识，提高回答准确性，较纯LLM提升40%+
- **智能体系统**：基于LangChain的Agent框架，能够根据用户问题自主决策调用工具
- **流式输出**：提供实时响应体验，平均响应时间<2秒
- **提示词工程**：根据场景动态调整提示词，提高模型输出质量

### 3. 性能优化
- **向量存储优化**：调整分块策略（chunk_size=800, chunk_overlap=80），添加缓存机制
- **模型调用优化**：为ChatTongyi模型添加参数调优（temperature=0.3, max_tokens=2000, top_p=0.9）
- **批量处理**：实现文档批量加载、分块和向量化，提高处理效率
- **缓存机制**：添加查询结果缓存，避免重复计算

### 4. 功能丰富性
- **智能问答**：基于RAG技术，结合文档知识提供准确回答
- **工具集成**：集成13+工具，包括产品比较、价格查询、购买链接生成、维护保养建议、故障排除指南、使用技巧等
- **多模态支持**：支持PDF、TXT文档解析和处理
- **外部数据集成**：可获取用户使用记录等外部数据

### 5. 用户体验优化
- **响应式设计**：适应不同设备屏幕
- **主题切换**：支持多种主题选择，包括深色模式
- **实时交互**：流式输出响应，提供实时反馈
- **智能提示**：常见问题快捷按钮，引导用户输入
- **动画效果**：添加消息动画，提升视觉体验

### 6. 配置管理与安全性
- **环境变量支持**：允许通过环境变量覆盖配置文件中的设置
- **配置验证**：添加配置验证机制，确保配置文件包含必要的键
- **错误处理**：完善的错误处理机制，提高系统稳定性
- **安全措施**：加强用户输入验证，防止注入攻击

### 7. 可扩展性
- **容器化部署**：支持Docker容器化部署
- **API集成**：可与其他系统集成
- **工具扩展**：易于添加新工具和功能
